# Sales Performance Dashboard (Power BI)

## Overview
This project is an interactive **Sales Performance Dashboard** built in **Power BI** to track sales, profit, orders, and profitability.  
It enables quick business insights using KPIs, trend analysis, category performance, and top products.

## Key Features
- **KPI Cards:** Total Orders, Profit Margin %, Total Sales, Total Profit
- **Trend Analysis:** Total Sales by Month
- **Category Analysis:** Total Sales by Product Category
- **Profitability Analysis:** Total Profit by Product Sub-Category (highlights loss-making sub-categories)
- **Top Products:** Top 5 Products by Total Sales
- **Filters:** Region slicer for interactive analysis

## Key Insights (Example)
- **Technology** is the highest revenue-generating product category.
- Sales show an upward trend towards the end of the year, indicating **seasonal demand**.
- Some sub-categories show **negative profit**, suggesting pricing/cost optimization opportunities.

## Tools Used
- Power BI Desktop
- DAX Measures
- Sample Superstore Dataset (Excel)

## Files in This Folder
- `dashboard.pdf` (exported report)
- `dashboard_screenshot.png` (preview image)

## How to Use
1. Open Power BI Desktop
2. Load the dataset (Orders table)
3. Create measures from `measures.txt`
4. Build visuals as shown in the exported PDF/screenshot
5. Use the Region slicer to explore data interactively
